# Фронтенд часть чата

### Описание проекта: 
Чат в реальном времени для общения! 

![alt text](http://url/to/Безымянный.png)
![alt text](http://url/to/Безымянный1.png)

### Технологии:
+ React.js
+ Socket.io
+ Node.js
+ Express.js

### Функционал: 

+ Обмен сообщениями
+ Отправка смайликов
+ Вход в комнату
+ Выход из комнаты


### Запуск проекта локально

`npm start` — запускает приложение в браузере
